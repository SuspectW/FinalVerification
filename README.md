# FinalVerification
## Задача: 
### Написать программу, которая из имеющегося массива строк формирует новый массив из строк, длина которых меньше, либо равна 3 символам.
## Описание алгоритма решения:
### Решение дается через заданный на старте массив. Сначала объявляются два массива: изначальный и второй такой же длины. Далее идет метод, внутри цикла которого проверяется условие на соответствие необходимой длины. Если выполняется, элемент первого массива заносится в count элемента второго массива. После присвоения увеличивается переменная count на 1 и возвращается к циклу for, в котором i увеличивается на 1. И так проверяется до конца.
